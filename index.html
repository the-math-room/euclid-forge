Euclid Forge Click empty space: add point Drag point: move point Drag triangle: move triangle Shift-click points: select T: create triangle from 3 selected points G: create centroid for selected triangle M: create/reuse side segments and midpoints H: hide selected nodes U: unhide all hidden nodes Arrow keys: pan viewport + / -: zoom viewport [ / ]: rotate viewport \\: reset viewport rotation 0: reset viewport Ctrl/Cmd+Z: undo Ctrl/Cmd+Shift+Z or Ctrl/Cmd+Y: redo
